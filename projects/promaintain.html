Custom maintenance management portal

The brief

A UK facilities management company handling 200+ commercial properties needed a custom web application to replace their spreadsheet-based workflow. Their clients needed to log maintenance requests, track job status, and download reports.

What I built

  • Laravel backend with RESTful API
  • React SPA dashboard for clients and admins
  • Job ticketing system with priority levels and auto-assignment
  • Real-time status updates via WebSockets
  • Automated PDF report generation (weekly/monthly)
  • Email notifications at every workflow stage
  • Role-based access (admin, manager, client, engineer)

Tech stack

Laravel 11, React 18, Inertia.js, PostgreSQL, Laravel Echo, DomPDF, Tailwind CSS

Result

Job completion time reduced by 40%. Client satisfaction score went from 3.2 to 4.8. 200+ properties managed through the system within 3 months of launch.
